Key results show this effective Bethe ansatz yields high-quality approximate eigenstates for weakly broken integrability but degrades for strong breaking, making it both a practical solver and a diagnostic tool. Its significance lies in providing a new, structured representation for nearly integrable states, allowing one to leverage algebraic techniques beyond the strict limits of integrability.|

The key result is that, unlike the Fréchet distribution found in the Lieb-Liniger model, these matrix elements follow a generalized inverse Gaussian distribution. This finding is significant as it reveals a new universality class for operator statistics, advancing our understanding of the Eigenstate Thermalization Hypothesis in integrable systems.|

This work is significant as it provides new, exact hydrodynamic models for systems with nonrelativistic scale invariance, offering insights into extreme transient states in such fluids.|
